====
Plan
====

To switch to Plan, players take the following steps after Learn.

#.  Select **Plan** to go to **Plan** interface.

#.  In the **Plan** interface:
    * Read **Practice Lab Goals**.
    * Read **DIY Goals**.
    * Select **Practice** to go to **Practice** interface.
